# 🎓 College Lab Assignments

This repository contains my laboratory programs for the current semester, covering **Core Java**, **Advanced JSP Web Development**, and **Python Scripting**.

---

## 📂 Navigation Guide

To help you find the source code quickly, please follow these directory paths:

> [!IMPORTANT]
> **For Standard Java Programs:**
> Follow `<program_name>` ➔ `src` ➔ `App.java`
> 
> **For JSP Web Applications:**
> Follow `<program_name>` ➔ `src` ➔ `main` ➔ `webapp` ➔ `index.jsp`
